Detailed Description

Formats a query result to KML.

Attributes will be stored as KML ExtendedData. Hierarchies of XML data will be stored as KML attribute names. KML requires that styles are formatted within the dataset, therefore this class has to utilize the KMLStyleFormatter implementation.
